*******************
Django Filter Sort
*******************

Django Filter Sort is a simple Django app to create class view with sorting and ordering.

Documentation 
=============

You can find the documentation at the following [link](https://django-filter-sort.readthedocs.io/en/latest/)

Quick start
===========

You can install it from Pypi in any way do you want
Ad example

    pip install django-filter-sorter

Or
    
    pipenv install django-filter-sorter
